📝 Como obter seu GitHub Token:
1. Acesse GitHub Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
2. Clique em "Generate new token" → "Generate new token (classic)"
3. Dê um nome (ex: "Expense Tracker")
4. Marque o escopo: repo (acesso completo ao repositório)
5. Clique em "Generate token" e copie o token gerado

⚠️ Importante: O token ficará salvo apenas no seu navegador (localStorage).
